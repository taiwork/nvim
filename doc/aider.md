
## install

Following the [Installation | aider](https://aider.chat/docs/install.html) to install the aider.

```bash
curl -LsSf https://aider.chat/install.sh | sh
export OPENROUTER_API_KEY=<your_api_key_value>
```

## Usage

```bash
aider --model openrouter/google/gemini-2.5-pro-exp-03-25:free
```
